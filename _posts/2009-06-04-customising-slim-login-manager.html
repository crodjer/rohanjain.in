Installing SLiM on Debian/Ubuntu simple as sudo apt-get install slim

I take no credit for the background. It comes from my friend's collection of wallpapers. The image for the panel is borrowed from the debian-moreblue theme, which is provided by default. The only thing I worked on is playing around with the value of the coordinates of the panel and with the fonts. Creating a theme for SLiM is ridiculously simple.

Note: All the above files should be located in /usr/share/themes/theme-name, where theme-name should be replaced by any name that you want. For example in my case it is /usr/share/themes/yeban

There are essentially three parts of a slim theme.
  1. A background image. The image file could be JPEG or PNG and should be named 'backgound.jpg' or 'background.png' depending on the image format you have.
  2. An image to be used as the panel. Panel is the area where you enter your username and password. The image file could be JPEG or PNG and should be named 'panel.jpg' or 'panel.png' depending on the image format you have.
  3. A configuration file which tells slim about the placement of the panel, the input boxes and the font to be used. This file is to be named 'slim.theme'. Here is mine.




As you can see the settings are key-value pairs and the name of the keys are self explanatory. You can find a complte listing and explanation of the settings here.
To put into effect the new theme you just created
  1. Open the file /etc/slim.conf
  2. Go to the custom_theme setting and change the default value to your theme-name
  3. Save the file and then login and logout to see the effect.
You can download my theme from git-hub(zipped file), change the name to whatever you want and use it.
