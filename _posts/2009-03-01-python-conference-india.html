Python enthusiasts in India, intend to have an Indian edition of the PyCon (a global confernce), in association with BangPypers (the Bangalore Python Users Group). The central idea is "a get together of the pythonistas from around the country to have a good fun pythonic time." The PyCon aims to provide (also one can read the Proposal Draft):
  • a rallying point for big/small Python groups in India to work together.
  • exposure to open source Python hackers and newbies in India.
  • to make oppurtinities for talented programmers to contribute to Python and its related projects.
  • spread the Python spirit to the Corporate India and College students alike.
A general outline of the event is something like this:
  • Tutorials
  • General Talks
  • Workshops
  • Lightning Talks
  • Programming Sprints
The event is yet to be planned, and, any suggestions about venue, timing, budget, sponsoring, publicity, or making the website can be posted at the PyCon wilki. The suggested dates are of September which makes me apprehensive about active participation of college students, for whom summers would probably be the best time.

Let us all pledge to make the event a grand success.
